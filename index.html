Frequently asked questions

What is an MBOX file?

MBOX (mailbox) is a standard format for storing email messages. Many email clients use it — including Gmail (via Google Takeout), Mozilla Thunderbird, Apple Mail, and Eudora. Each message is stored sequentially in a single .mbox file.

How do I export my Gmail emails as MBOX?

Go to myaccount.google.com → Data & privacy → Download your data (Google Takeout). Select only Mail, choose .mbox format, and download. The exported file can be used directly with this tool.

What columns are included in the CSV?

By default the CSV contains: Date, From, To, CC, BCC, Subject, Body, and Message-ID. You can choose which columns to include before downloading using the column selector.

Is my email data private?

Yes — completely. This tool runs entirely in your browser using JavaScript. Your MBOX file is read locally and never sent to any server. We have no access to your emails whatsoever.

How large a file can I convert?

This depends on the memory available in your browser. Files up to ~500 MB work well on most modern computers. For very large exports (multi-GB), we recommend splitting the MBOX file first.

Does it handle attachments?

The CSV includes the plain-text body of each email. Attachments are not extracted — they remain in the original MBOX file. The number of attachments is not currently included as a column, but this is on our roadmap.

Why open the CSV in Excel and see garbled characters?

The CSV is exported as UTF-8 with a BOM (byte order mark) to signal the encoding. If Excel still shows garbled text, use Data → From Text/CSV and select UTF-8 encoding during import.
